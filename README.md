# Monte Carlo Simulation For Computational Finance 
## Srinjoy Som 
### Topics Covered 
`Monte_Carl.ipynb`
1- Linear Congruence generator and Inverse transform method \
2 - Generating samples from N(0 , 1 ) using BOX-Muller and Marsaglia-Bray Method \
3 - Plotting and Visualization 

`MCF.ipynb` \
1-  Generating Brownian Motaion Path \
2 - Simualting Stock Price from Bond price Data \
3 - Jump diffusion Model \
4- Low Discrepency Point Sequence (Van-der-corput and Halton) \
5 - Call option Pricing , Up and Out option pricing \
6 - Call option Pricing using control variate \
7 - Up and out option pricing using antithetic variable 

`PDE.ipynb` \
1- Europian option pricing by taking discrete time steps under binomial model \
2- Europian option priciing by solving PDE under Bsm model \
3 - American option pricing by cryers algorithm
4 - Option Pricing by Approximation Based on Interpolation 
5 - Option pricing by Quadratic approaximation
